# 简单 Skill 模板

```text
Skill 名称：
[给 skill 起名]

触发条件：
- 当我需要 [场景] 时使用

输入：
- 主题：
- 当前水平：
- 时间预算：
- 约束：
- 期望输出：

执行规则：
1. 先确认目标和边界
2. 如果上下文不足，先提 1 到 3 个澄清问题
3. 按“拆解 -> 执行 -> 检查 -> 总结”的顺序工作
4. 输出中必须包含验证动作
5. 遇到高风险或越界内容，停止并给出安全替代方案

输出模板：
1. 目标理解
2. 执行步骤
3. 结果
4. 验证清单
5. 下一步建议

边界说明：
- 不假装已知未知信息
- 不跳过验证
- 不处理未授权或攻击性内容
```
