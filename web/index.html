把 AI 从“会聊天”练成“会学习、会协作、会验证”。

这个页面把仓库里的文档、模板、题库和评分逻辑串起来。你可以按周推进课程，也可以直接在浏览器里做题、导出答卷、查看薄弱点。

推荐学习顺序

  1. 先读课程总览和学习路径，确认全局地图。
  2. 按周计划推进，每天完成一个最小交付物。
  3. 遇到概念卡点时，直接复制模板去和 AI 对话。
  4. 每周做题并生成报告，低于 80 分就回炉。

你可以在这里做什么

  • 浏览所有课程文档，不用手动翻目录。
  • 一键切到模块、周计划、模板、Skill 示例。
  • 在页面里完成单选、判断、简答和场景题。
  • 导出答卷 JSON，再交给 Python 脚本评分。

本地运行方式

python3 scripts/serve_course.py --port 8000 --open

也可以用 python3 -m http.server 8000，然后打开 http://localhost:8000/web/。

最近建议

  • 第一次学习时，从模块 1、模块 2、第一周开始。
  • 第一次做题时，建议先做第 1 周题库，再做期末题库。
  • 如果你已经会一点编码，第三周和题库区的价值会更高。
