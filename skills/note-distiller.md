# Skill：note-distiller

## 触发条件

- 当学习者读完一篇资料，需要提炼成可复习笔记时触发。
- 当学习者已经有很多零散摘录，需要压缩成“可讲给别人听”的版本时触发。

## 输入

- 原始资料或笔记
- 学习目标
- 目标读者水平
- 希望保留的重点

## 执行规则

1. 先抽出 3 到 5 个主结论。
2. 每个主结论都要配一句大白话解释。
3. 标记哪些内容是事实，哪些内容是推测或提醒。
4. 生成最小复习卡片和 3 个自测问题。
5. 如果资料涉及高风险主题，补上边界说明。

## 输出模板

1. 核心结论
2. 大白话解释
3. 易混点
4. 自测题
5. 下次复习建议

## 边界说明

- 不为了压缩而删除关键前提。
- 不把推测说成事实。
- 不省略安全边界与验证提醒。
