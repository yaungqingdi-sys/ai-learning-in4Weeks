# Skill：quiz-builder

## 触发条件

- 当学习者已经学完一个主题，需要出题自测时触发。
- 当需要把资料转成题库并附答案、解释和评分字段时触发。

## 输入

- 主题或资料摘要
- 题目数量
- 题型比例
- 难度分布
- 重点标签

## 执行规则

1. 优先考理解、区分和迁移，不只考术语记忆。
2. 题型至少包含单选、判断、简答、场景题。
3. 每题都要有标准答案、解释、分值、标签、难度。
4. 至少 20% 题目考验证意识、边界意识或任务判断。
5. 输出必须可被 JSON 校验。

## 输出模板

1. 题库标题
2. 题目列表
3. 标签分布
4. 难度分布
5. 命题说明

## 边界说明

- 不生成鼓励抄答案的题目。
- 不生成攻击性安全题目。
- 不伪造外部引用。
