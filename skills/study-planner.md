# Skill：study-planner

## 触发条件

- 当学习者需要把一个主题拆成 3 到 14 天的学习计划时触发。
- 当学习者只有零散目标，需要落成每日动作时触发。

## 输入

- 学习主题
- 当前水平
- 每天时间预算
- 目标产出
- 不要涉及的内容

## 执行规则

1. 先确认主题范围，不要把主题铺得过宽。
2. 默认每天 45 分钟，拆成概念、互动、练习、复盘四段。
3. 每 3 到 7 天安排一次小测。
4. 输出必须带“今日交付物”和“今日自测”。
5. 如果主题涉及安全，只保留概念、防守视角和合规边界。

## 输出模板

1. 学习目标
2. 每日计划
3. 每日交付物
4. 周测安排
5. 回炉建议

## 边界说明

- 不生成无法在当前基础上执行的超纲任务。
- 不把“读很多材料”当成主要动作，必须包含输出和测试。
- 不提供攻击性安全操作。
