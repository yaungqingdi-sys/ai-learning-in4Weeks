# 参考资料

- [Google AI Studio](https://ai.google.dev/aistudio)：用来实际选择 Gemini 模型，完成大纲确认、逐页细化、ASCII 布局和 SVG 生成。
- [Gemini 模型总览](https://ai.google.dev/models/gemini)：用来确认当前可用模型名称与能力边界，避免在流程里写错模型或高估能力。
- [Prompt design strategies](https://ai.google.dev/guide/prompt_best_practices)：用来看“先澄清再生成”“分阶段确认”的 Prompt 设计原则。
- [Google AI Studio Build mode](https://ai.google.dev/gemini-api/docs/aistudio-build-mode)：用来理解 Preview 面板和结果预览方式，适合对应 SVG 确认环节。
- [SVG 2](https://www.w3.org/TR/SVG2/)：用来补 SVG 作为文本图形格式的正式定义，理解为什么它适合作为中间可审查格式。
- [python-pptx](https://github.com/scanny/python-pptx)：用来看 PowerPoint 文件结构与 Python 写入方式；如果要把 Gemini 生成的逐页备注写进 PPT，可从这里继续查。
- [svg-to-editable-ppt.zip](svg-to-editable-ppt.zip)：本模块附带的 Codex skill 压缩包，用来把 AI Studio 导出的 SVG 或包含 SVG 标记的 `.txt` 文件转成可编辑 PPT。
- [全局参考资料](../../../reference.md)：用来继续查课程范围内的官方入口。
