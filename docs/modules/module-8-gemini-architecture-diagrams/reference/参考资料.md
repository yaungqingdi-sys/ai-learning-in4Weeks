# 参考资料

- [Gemini 模型总览](https://ai.google.dev/models/gemini)：用来确认 Google 官方当前可用模型名称。本文提到的 Pro 系列，以这里列出的 `gemini-3-pro-preview` 为准。
- [Google AI Studio](https://ai.google.dev/aistudio)：用来进入 AI Studio 并直接试模型。
- [Google AI Studio Quickstart](https://ai.google.dev/tutorials/ai-studio_quickstart/)：用来看 AI Studio 里的 Chat prompt、多轮对话和参数面板怎么使用。
- [Google AI Studio Build mode](https://ai.google.dev/gemini-api/docs/aistudio-build-mode)：用来看右侧 Preview 面板如何显示生成结果。本文里“生成后点击 Preview 看结果”的环节，主要对应这里的工作方式。
- [Prompt design strategies](https://ai.google.dev/guide/prompt_best_practices)：用来补“先澄清再生成”的 Prompt 设计原则。
- [Image generation with Gemini](https://ai.google.dev/gemini-api/docs/image-generation)：用来了解 Gemini 系列在视觉生成和可视化迭代上的整体能力边界。
- [全局参考资料](../../../reference.md)：用来继续查课程范围内的官方入口。
